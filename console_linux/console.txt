- rm -r directory																			- удалить папки и их содержимое рекурсивно
	-f																							- игнорировать несуществующие файлы, никогда не запрашивать
	-v																							- объяснить что делается
- ls -a																						- аналог dir, но показывает все скрытые папки
- ls -ls																						- как предыдущий, но все в 1 столбик
- mv file.txt file.html																	-переименовать file.txt в file.html. На самом деле переносит файл и переименовывает
- find . *.php																				- отыскать все php файлы в действующей директории и поддиректории
- > text.txt																				- создать пустой текстовый файл
- touch text																				- создаст пустой файл text
- bc																							- запустить консоль как калькулятор
- gvfs-open <file>																		- запустить файл
- which <program>																			- показать где расположена установленная программа
